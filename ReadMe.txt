RECOMMANDATION FOR WINDOWS USERS
We really suggest you to follow this tutorial to activate colors for your Windows terminal !
LimitSwap now uses colors to show you errors, infos and success

1/ Launch "Regedit"
2/ Go to [HKEY_CURRENT_USER \ Console]
3/ Right click on white space / create new DWORD 32 bit key called "VirtualTerminalLevel" and set it to 1
4/ Enjoy
